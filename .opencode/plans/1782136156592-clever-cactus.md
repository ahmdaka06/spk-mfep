## MFEP Method Summary

**5 Criteria with Fixed Weights:**
| # | Criteria | Weight (FW) |
|---|----------|-------------|
| 1 | Kedisiplinan (Discipline) | 0.3 |
| 2 | Kualitas Kerja (Work Quality) | 0.2 |
| 3 | Tanggung Jawab (Responsibility) | 0.2 |
| 4 | Kerja Sama Tim (Teamwork) | 0.2 |
| 5 | Loyalitas dan Sikap (Loyalty) | 0.1 |
|   | **Total** | **1.0** |

**Formulas:**
- WE (Weighted Evaluation) = FW × E
- Total Score = Σ(WE per criterion)
- E = Evaluation value (scale 1-100)

## System Workflow

1. Admin prepares employee data + evaluation scores (via Excel import or manual)
2. Admin manages criteria and weights
3. System calculates MFEP: WE = FW × E for each criterion
4. System sums WE values to get Total score
5. System ranks employees by Total score
6. Kepala OPD views ranking and determines exemplary employee
7. Generate SK (Surat Keputusan) / Report

## 2. MFEP Method (Fixed Weights)

| # | Criteria | Weight (FW) |
|---|----------|-------------|
| 1 | Kedisiplinan | 0.30 |
| 2 | Kualitas Kerja | 0.20 |
| 3 | Tanggung Jawab | 0.20 |
| 4 | Kerja Sama Tim | 0.20 |
| 5 | Loyalitas dan Sikap | 0.10 |

**Formula:** `Total = (Kedisiplinan × 0.30) + (Kualitas × 0.20) + (Tanggung × 0.20) + (Kerja Sama × 0.20) + (Loyalitas × 0.10)`
